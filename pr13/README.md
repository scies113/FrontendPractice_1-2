# Практическая работа №13: Service Worker и кэширование (PWA)

## Описание
создание офлайн-приложения «Заметки» с использованием Service Worker для кэширования ресурсов. приложение позволяет добавлять, удалять и просматривать заметки, сохраняя данные в LocalStorage. Service Worker обеспечивает работу приложения без подключения к интернету.

## Что реализовано
- **Service Worker** — установка, активация, кэширование статических ресурсов (HTML, CSS, JS, manifest)
- **Стратегия кэширования** «Cache First, Then Network» — сначала ответ из кэша, при отсутствии — запрос к сети с последующим кэшированием
- **Очистка старых кэшей** при обновлении версии Service Worker
- **LocalStorage** — хранение заметок между сессиями
- **Индикатор онлайн/офлайн** — отслеживание статуса подключения через `navigator.onLine`
- **Защита от XSS** — экранирование HTML при выводе пользовательского текста
- **SCSS** — переменные, миксины для кнопок, вложенность, анимации, адаптив (media query)
- **PWA manifest.json** — название, тема, режим отображения `standalone`

## Технологии
- HTML5 + CSS3 (SCSS/SASS)
- JavaScript (ES6+)
- Service Worker API
- Cache API
- LocalStorage API
- PWA (Progressive Web App)
- npm (скрипты сборки)

## Запуск
1. Переходим в папку проекта:
   ```
   cd pr13
   ```
2. Устанавливаем зависимости:
   ```
   npm install
   ```
3. Запускаем локальный сервер:
   ```
   npm run dev
   ```
4. Открываем браузер по адресу `http://127.0.0.1:8080/`
