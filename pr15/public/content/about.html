О приложении

Версия 1.0.0

Это приложение для заметок, построенное на архитектуре App Shell.

Каркас приложения (HTML, CSS, JS) кэшируется при первом посещении и загружается мгновенно.

Динамический контент (страницы «Главная» и «О приложении») подгружается внутрь контейнера.

Технологии

  • HTML5 + CSS3 (SCSS)
  • JavaScript (ES6+)
  • Service Worker API
  • Cache API (два кэша: статика + динамический контент)
  • LocalStorage API
  • HTTPS (самоподписанный сертификат mkcert)
  • PWA (Progressive Web App)

Стратегии кэширования

  • Cache First — для статических ресурсов (CSS, JS, иконки)
  • Network First — для динамических страниц (/content/*.html)
