О приложении

Версия 1.0.0

Приложение для заметок с поддержкой WebSocket и Push-уведомлений.

WebSocket обеспечивает мгновенные уведомления в реальном времени между вкладками браузера.

Push-уведомления работают даже при закрытой вкладке (через Service Worker).

Технологии

  • HTML5 + CSS3 (SCSS)
  • JavaScript (ES6+)
  • Socket.io (WebSocket)
  • Web Push API (VAPID)
  • Service Worker API
  • Node.js + Express (бэкенд)
  • LocalStorage API

Как это работает

  • WebSocket — двустороннее соединение, сервер отправляет события клиентам мгновенно
  • Push — сервер отправляет push-уведомление через Web Push API
  • Service Worker — обрабатывает push-события и показывает системные уведомления

Как протестировать

  • Откройте две вкладки с приложением
  • В одной вкладке добавьте заметку
  • Во второй появится всплывающее уведомление (WebSocket)
  • Если включены Push — придёт системное уведомление
