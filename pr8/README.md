# Практическая работа №8: JWT Авторизация

## Описание
реализация системы аутентификации на основе токенов(JWT). Защита маршрутов API.

## Технологии
- Node.js + Express
- jsonwebtoken (создание и верификация токенов)
- bcrypt (хеширование паролей)
- Swagger (документация)

## Запуск
1. cd back(перейти в папку back(../pr8/back))
2. npm install(зависимости)
3. npm run dev(заупск)
4. окрыть через: http://localhost:3000/api-docs

## Сценарий работы программы
1. POST /api/auth/register - регистрация пользователя
2. POST /api/auth/login - логинимся в системе и получаем токен
3. используем токен в заголовке Authorization: Bearer <token> для доступа к:
   - GET /api/auth/me (а также - POST /api/products).
4. проверяем жизнь токена в - GET /api/auth/me (30 секунд, после чего последует 401 ошибка)
