# Практическая работа №17: Детализация Push — Напоминания

## Описание
Доработка приложения заметок с добавлением **напоминаний** — push-уведомлений, запланированных на определённую дату и время. Сервер хранит активные таймеры в `Map` и отправляет push в нужный момент. В уведомлении есть кнопка **«Отложить на 5 минут»** (snooze), которая переносит напоминание.

## Что реализовано
- **Форма напоминаний** — текст + дата/время (`datetime-local`), отправка на сервер через WebSocket
- **Планировщик на сервере** — `Map` с активными таймерами, `setTimeout` для отложенной отправки push
- **Push с кнопками** — уведомление содержит action «⏰ Отложить на 1 минуту»
- **Snooze** — клик по кнопке отправляет `POST /snooze`, сервер отменяет старый таймер и ставит новый на 1 минуту
- **WebSocket** — real-time уведомления между вкладками (обычные задачи + напоминания)
- **Service Worker** — обработка push с кнопками, обработка `notificationclick` с action
- **App Shell** — вкладки «Главная» и «О приложении»
- **LocalStorage** — заметки с полем `reminder` (timestamp)

## Технологии
- HTML5 + CSS3 (SCSS/SASS)
- Vanilla JavaScript (ES6+)
- Socket.io (WebSocket)
- Web Push API (VAPID)
- Service Worker API (push + notification actions)
- Node.js + Express (бэкенд с планировщиком)
- LocalStorage API
- npm


## Запуск

### 1. Установка зависимостей
```
cd pr17
npm install
```

### 2. Запуск сервера
```
npm run dev
```
Сервер запустится на `http://localhost:3002`

### 3. Открыть приложение
Откройте браузер по адресу `http://localhost:3002/`

### 4. Тестирование напоминаний
1. Включите push-уведомления (кнопка в футере)
2. Заполните форму «Добавить с напоминанием»: текст напоминания
3. Нажмите **«⏰ Добавить с напоминанием»** — push придёт через 1 минуту
4. Нажмите **«⏰ Отложить на 1 минуту»** — напоминание придёт снова через 1 минуту


## Что заснять на видео (подробно)

### 1. Запуск сервера
- Показать терминал с командой `npm run dev`
- Показать сообщение: `Сервер запущен на http://localhost:3002`

### 2. Открытие приложения
- Открыть `http://localhost:3002/`
- Показать страницу: заголовок «📝 Заметки — Напоминания», две формы (обычная + с напоминанием)

### 3. Обычная заметка
- Добавить заметку через первую форму
- Показать, что она появилась в списке

### 4. Заметка с напоминанием
- Заполнить форму «Добавить с напоминанием»:
  - Текст: «Позвонить врачу»
- Нажать **«⏰ Добавить с напоминанием»**
- Показать заметку в списке — под текстом видно `⏰ Напоминание: дата и время`

### 5. Push-уведомление
- Включить push-уведомления (кнопка «🔔 Включить уведомления» в футере)
- Подождать 1 минуту — придёт push-уведомление
- Показать, что в уведомлении есть кнопка **«⏰ Отложить на 1 минуту»**

### 6. Snooze (отложить)
- Нажать **«⏰ Отложить на 1 минуту»** в уведомлении
- Показать терминал сервера — сообщение: `Напоминание "..." отложено на 1 минуту`
- Через 1 минуту придёт новое push-уведомление: `Напоминание отложено`

### 7. WebSocket между вкладками
- Открыть вторую вкладку
- В первой добавить напоминание — во второй появится всплывающее уведомление `⏰ Напоминание: ...`

### 8. Service Worker
- DevTools → **Application** → **Service Workers** — статус `activated and is running`
- DevTools → **Application** → **Cache Storage** — показать кэши:
  - `notes-cache-v5` (статика)
  - `dynamic-content-v3` (динамический контент)

### 9. LocalStorage
- DevTools → **Application** → **Local Storage** → `http://localhost:3002`
- Показать ключ `notes` — массив объектов с полями `id`, `text`, `reminder`

### 10. Страница «О приложении»
- Нажать вкладку **«О приложении»**
- Показать описание технологий: планировщик, push с кнопками, snooze

### 11. Офлайн-режим (опционально)
- DevTools → **Network** → **Offline** → обновить страницу
- Приложение загрузилось из кэша
