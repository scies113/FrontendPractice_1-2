О приложении

Версия 1.0.0

Приложение для заметок с поддержкой напоминаний и детализированных Push-уведомлений.

Можно создать заметку с напоминанием на определённую дату и время. Сервер запланирует push-уведомление.

В уведомлении есть кнопка «Отложить на 5 минут» — snooze.

Технологии

  • HTML5 + CSS3 (SCSS)
  • Vanilla JavaScript (ES6+)
  • Socket.io (WebSocket)
  • Web Push API (VAPID)
  • Service Worker API (push + notification actions)
  • Node.js + Express (бэкенд с планировщиком)
  • LocalStorage API

Как это работает

  • Форма напоминаний — текст + дата/время, отправляется на сервер
  • Планировщик — сервер использует setTimeout для отложенной отправки push
  • Push с кнопками — уведомление содержит action «Отложить на 5 минут»
  • Snooze — клик по кнопке отправляет POST /snooze, сервер переносит напоминание

Как протестировать

  • Включите push-уведомления кнопкой в футере
  • Создайте заметку с напоминанием на 1–2 минуты вперёд
  • Дождитесь push-уведомления
  • Нажмите «Отложить на 5 минут» — напоминание придёт снова
