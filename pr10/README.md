# Практическая работа №10: Хранение токенов и Refresh Flow

## Описание
реализация фронтенда на React с автоматическим обновлением JWT токенов.
используется Axios Interceptors для перехвата ошибки 401 и бесшовной ротации токенов.

## Технологии
- React + Vite
- Axios (Interceptors)
- LocalStorage
- Node.js + Express (Backend с Refresh-токенами)

## Запуск
1. Бэкенд:
   переходим в папку back(cd back)
   npm install(зависимости)
   node app.js(запуск)

2. Фронтенд:
   переходим в папку client(cd client)
   npm install(зависимости)
   npm run dev(запуск)

## Как протестировать обновление токена
1. войти в систему.
2. открыть инструменыт разработчика -> Network.
3. нажать кнопку "Симулировать истечение токена".
4. наблюдать цепочку: запрос -> 401 -> /refresh (200) -> повтор запроса (200).
